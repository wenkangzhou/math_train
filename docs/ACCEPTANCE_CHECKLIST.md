# 数数小火车验收清单

这份清单用于每次发布前做一轮“像小朋友一样使用”的验收。重点设备是 iPad 横屏，同时覆盖 iPad 竖屏、手机和 PC。

建议每轮验收使用生产构建：

```bash
pnpm build
pnpm preview --host 0.0.0.0
```

在同一局域网内，用 iPad / 手机访问电脑显示的地址。离线验收请先在线打开一次应用，等待页面显示“已可离线”或完成加载后再断网。

## 1. 基础健康检查

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| 类型检查 | 执行 `pnpm lint` | 无 TypeScript 错误 |
| 单元测试 | 执行 `pnpm test` | 全部测试通过 |
| 生产构建 | 执行 `pnpm build` | `dist` 和 `dist/sw.js` 正常生成 |
| 首屏加载 | 打开生产预览地址 | 无白屏、无明显报错、首页内容完整 |

## 2. iPad 横屏核心验收

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| 首屏重点 | iPad 横屏打开首页 | 小朋友第一眼能看到“开始做题”，按钮足够大、颜色突出 |
| 首页入口 | 查看“我的小火车”“长期错题本”“答题记录” | 文案不换行到难读状态，不被裁切 |
| 设置折叠 | 点击“请大人调整任务” | 设置能展开/收起；默认不干扰孩子开始 |
| 开始练习 | 点击“开始做题” | 进入答题页，题卡、进度条、数字键盘都在合理位置 |
| 一屏可用 | 在答题页点“帮帮我” | 辅助内容尽量在一屏内，无需孩子频繁滑动才能理解 |
| 数字键盘 | 依次点击数字、清除、确定 | 触控面积足够，不误触；选中态清晰 |
| 进度条 | 完成 2～3 题 | 小火车/站点/当前进度能看懂，文字不太小 |

## 3. iPad 竖屏与手机验收

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| iPad 竖屏首页 | 竖屏打开首页并滚动 | 无横向滚动；底部开始按钮可见、可点击 |
| iPad 竖屏答题 | 开始一趟 10 题 | 题卡、辅助、键盘纵向排列合理，不遮挡 |
| 手机首页 | iPhone / Android 手机打开首页 | 无横向滚动；主按钮、入口卡片、文字不被裁切 |
| 手机答题 | 完成 2～3 题 | 数字键盘不拥挤；辅助面板可读 |
| 安全区 | iPhone 全屏 / iPad 主屏模式打开 | 底部按钮不被 Home Indicator 遮挡 |

## 4. 语音与音效验收

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| 自动朗读开 | 保持“自动朗读题目”开启，点击开始 | 第一题能朗读；切到下一题后能继续朗读 |
| 自动朗读关 | 关闭“自动朗读题目”，重新开始 | 第一题和后续题都不自动朗读 |
| 手动朗读 | 答题页点击“朗读” | 能读当前题；故事题优先读故事文本 |
| 反馈语音 | 打开“朗读答题反馈”，答对/答错 | 有温和语音反馈，不与题目朗读严重重叠 |
| 音效 | 打开音效后答对/答错 | 答对、答错、发车/到站有明显但不刺耳的声音 |

> 注意：Web Speech API 在不同 iPadOS / 浏览器上的表现差异较大。语音必须用真机听一次，尤其是 iPad Safari 和微信内置浏览器。

## 5. 离线验收

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| 首次缓存 | 在线打开生产地址一次 | 页面正常加载，离线状态不报错 |
| 断网刷新 | 打开飞行模式后刷新页面 | 应用仍能打开首页 |
| 离线练习 | 断网状态完成 3～5 题 | 答题、提示、奖励本地可用 |
| 主屏冷启动 | 添加到主屏后，断网关闭再打开 | 能进入应用，不白屏 |
| 恢复在线 | 关闭飞行模式重新打开 | 数据仍保留，页面正常 |

## 6. 奖励系统验收

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| 星星奖励 | 完成一趟含简单题和难题的练习 | 首次答对按难度给 1～5 星；重试答对给 1 颗坚持星 |
| 到站奖励 | 完成一趟后看结果页 | 有到站/路线/邮票反馈，孩子能感到“完成一趟旅程” |
| 车库详情 | 打开“我的小火车”，点击机车 | 能查看详情；已解锁/未解锁状态清晰 |
| 旅行动画 | 在车库点击“长途出发” | 跑动有站点、牧场、河桥、小镇等旅行感；结束回到原站 |
| 下一辆目标 | 查看下一辆机车提示 | 显示还差多少星、约再跑几趟，不制造过强挫败感 |

## 7. 错题本与间隔复习验收

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| 记录错题 | 故意答错一题，再重试答对 | 该题进入长期错题本；记录曾答答案和尝试次数 |
| 当天不提前掌握 | 当天再次练同一道错题 | 不应因为同一天答对多次就直接掌握 |
| 隔天复习 | 调整系统日期或等待隔天后打开 | 错题显示为可复习 |
| 三阶段推进 | 依次在隔天、3 天后、7 天后答对 | 通过 3 关后才进入“已掌握” |
| 复习失败 | 在复习时答错 | 复习阶段回退，下一次从隔天重新开始 |

## 8. 自适应难度验收

| 项目 | 步骤 | 通过标准 |
| --- | --- | --- |
| 默认可控 | 不开启自动难度 | 练习范围按家长设置，不自动变化 |
| 表现很好 | 开启自动难度，连续两趟高正确率、少提示 | 下一轮难度温和提升 |
| 表现吃力 | 开启自动难度，低正确率、多次重试 | 下一轮难度降低或回到更稳的题型 |
| 错题复习隔离 | 只做错题本复习 | 不改变常规练习难度 |

## 9. 儿童可理解性验收

找一位 4～7 岁小朋友，尽量不提前解释，只说“你可以试着让小火车出发”。

| 观察点 | 通过标准 |
| --- | --- |
| 10 秒内是否知道点哪里开始 | 能主动点“开始做题”，或明显被按钮吸引 |
| 是否理解数字键盘 | 能自己选择答案并点“确定” |
| 答错后是否焦虑 | 能接受提示继续尝试，没有强挫败感 |
| 是否会使用帮助 | 点“帮帮我”后能理解图片/摆一摆/数轴之一 |
| 奖励是否有动力 | 完成一趟后愿意看小火车或再跑一趟 |
| 家长是否能理解记录 | 家长能看懂答题记录、错题本、下一次复习时间 |
